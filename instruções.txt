Nesta semana, vamos exercitar os conceitos: Frameworks do lado do servidor.

Comece escolhendo um tema (Temas).

Crie um projeto web utilizando um framework no lado de servidor que inclua:

   1.Criar rotas para /, /sobre, /seu_problema e /resposta_problema. OK

    2.A rota /seu_problema deve ser uma página com um formulário HTML de acordo com o seu tema.
    
    3.A rota /resposta_problema deve receber os dados do formulário, calcular e mostrar a resposta do problema.

    4.Implementar um middleware para registrar data, hora e rota de cada solicitação.

    5.Adicionar um middleware para lidar com erros e rotas não definidas (Erro 404).

    6.Disponibilize seu projeto em um repositório público. OK
